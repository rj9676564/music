Molten Music API

欢迎使用 Molten Music 后端 API。本系统为播客管理和 AI 辅助收听提供强大的底层支持，包括播客订阅解析、音频流媒体服务、基于 Whisper 的自动转录以及基于大模型的智能摘要生成。

基础 URL
http://localhost:8080
默认运行端口为 8080，支持 CORS 跨域请求。

频道管理

GET /api/channels
获取所有已订阅的播客频道列表。
Response
[
  {
    "id": "the-daily",
    "name": "The Daily",
    "author": "The New York Times",
    "rss": "https://feeds.simplecast.com/54nAGcIl",
    "updated_at": "2024-03-20T10:00:00Z"
  }
]
GET /api/channels/:id/episodes
获取指定频道的节目列表。支持智能刷新逻辑：超过 1 小时未更新时会自动同步 RSS。
参数类型必须描述
idstring是频道 ID
refreshquery否传 true 强制强制刷新 RSS

音频处理

POST /api/download
将播客音频下载到本地缓存服务器。
Request Body
{
  "guid": "episode-guid-123",
  "url": "https://example.com/audio.mp3"
}
GET /media/:filename
流式播放已缓存的音频文件。支持 HTTP Range 请求，优化播放体验。

AI 转录与摘要

POST /api/transcribe
调用 Whisper AI 服务器进行语音转文字，生成 SRT 字幕。
Request Body
{
  "guid": "episode-guid-123",
  "audioPath": "media_cache/abc.mp3"
}
POST /api/upload-srt NEW
手动上传 SRT 字幕文件。适用于已有字幕或通过外部工具生成的场景。
Form 字段类型必须描述
guidstring是节目 GUID
filefile是SRT 格式文件
POST /api/summary
使用 LLM (如 GPT-4) 生成节目摘要。
Request Body
{
  "guid": "episode-guid-123",
  "srtContent": "1\n00:00:01,000 --> 00:00:04,000\nHello world...",
  "model": "gpt-4-turbo"
}
POST /api/queue-transcription NEW
异步转录队列任务。添加到队列后，后台会按顺序处理下载和转录。

环境配置

后端服务支持通过环境变量进行高度自定义配置。

变量名描述默认值
DB_TYPE数据库类型 (sqlite/mysql)sqlite
DB_PATHSQLite 文件存储路径data/molten.db
DB_DSNMySQL 连接字符串-
WHISPER_SERVER_URLWhisper API 服务器地址-
OPENAI_API_KEYLLM 摘要所需密钥-

数据库结构

系统使用 GORM 进行数据库管理。主要表结构如下：

Channels (频道表)
  • • ID (Primary Key)
  • • Name (频道名称)
  • • Author (作者)
  • • RSS (源地址)
Episodes (节目表)
  • • GUID (Primary Key)
  • • Title (标题)
  • • SrtContent (字幕内容)
  • • Summary (AI 摘要)
